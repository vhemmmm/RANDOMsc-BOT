## Clone this project

```bash
> git clone https://github.com/vhemmmm/RANDOMsc-BOT
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> pkg install unzip
> unzip RANDOMsc-BOT
> cd RANDOMsc-BOT
> bash install.sh
```

### Usage
```bash
> npm start
```
